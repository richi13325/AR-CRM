# Changes — active SDD changes

Each change has its own folder: `openspec/changes/<change-name>/`.
Completed changes are archived to `openspec/changes/archive/YYYY-MM-DD-<change-name>/`.

Workflow per change:
1. `exploration.md` (optional, from sdd-explore)
2. `proposal.md` (from sdd-propose)
3. `specs/<domain>/spec.md` (from sdd-spec)
4. `design.md` (from sdd-design)
5. `tasks.md` (from sdd-tasks)
6. `verify-report.md` (from sdd-verify, after sdd-apply)
